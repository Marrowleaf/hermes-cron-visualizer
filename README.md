# 🔧 hermes-cron-visualizer

A [Hermes Agent](https://hermes-agent.nousresearch.com/) skill for **cron-visualizer** (devops category).

## Installation

```bash
cd ~/.hermes/skills/devops
git clone https://github.com/Marrowleaf/hermes-cron-visualizer.git cron-visualizer
```

## Usage

Once installed, this skill is automatically detected by Hermes Agent.

```
skill_view(name="cron-visualizer")
```
